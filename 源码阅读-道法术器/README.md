#### 法（设计原理与规范）
- 发挥空间
- 约束条件
#### 术（技术）
- 分布式算法
- 虚拟机技术
- RPC协议
#### 器（环境与工具）
- Statistic 统计工具
- Translation 翻译工具
- SequenceDiagram 时序图工具

- 容器化
- 云原生
